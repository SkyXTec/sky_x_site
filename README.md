# SKY x - Site Institucional

Site institucional da SKY x implementado com base no design do Figma.

## 🚀 Tecnologias

- HTML5
- css3
- JavaScript (Vanilla)
- Google Fonts (Inter & Roboto Condensed)

## 📁 Estrutura do Projeto

```
sky_x_site/
├── index.html          # Página principal
├── styles.css          # Estilos css
├── script.js           # Scripts JavaScript
├── README.md           # Este arquivo
├── LICENSE             # Licença
```

## 🎨 Design

O design foi implementado com base no wireframe do Figma disponível em:
https://www.figma.com/design/jgqiT59teXeZS9B4oCqIz9/

### Características:

- Design moderno e minimalista
- Totalmente responsivo
- Navegação intuitiva
- Ícones de redes sociais
- Hero section com imagem de destaque
- Tipografia personalizada (Inter e Roboto Condensed)

## 🌐 Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Ou use um servidor local:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (com live-server)
   npx live-server
   ```

## 📱 Responsividade

O site é totalmente responsivo e se adapta a diferentes tamanhos de tela:
- Desktop (1440px+)
- Tablet (768px - 1200px)
- Mobile (< 768px)

## ⚠️ Notas

- As imagens são carregadas diretamente da API do Figma
- Os links estão configurados como âncoras para futuras seções
- O design segue fielmente o wireframe fornecido
